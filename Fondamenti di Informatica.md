

# Conversioni binarie

- Con *n* bit codifichiamo $2^n$ numeri nell'intervallo $[0; 2^n - 1]$
- Per convertire un numero naturale in binario si usa il metodo dei resti, dividendo il numero considerato per 2: 
		*- esempio:* 
		

| dividendo | divisore | resto |
| --------- | -------- | ----- |
| 19        | 2        | 1     |
| 9         | 2        | 1     |
| 4         | 2        |       |
